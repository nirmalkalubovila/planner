Features

Habit Tracker

Create daily habits with time slots. Set start dates. Automatic conflict prevention. Syncs with week planner.

Goal Tracker

Plan long-term goals with weekly breakdowns. Organize into collapsible years and months. Shift entire goals when needed.

Week Planner

Visual 24x7 grid for every hour. Paint goal work. Add custom events. Navigate through weeks with data persistence.

Easy Editing

Edit any habit or goal anytime. Update times and descriptions. Delete with confirmation. Cancel to discard changes.

Week-Based System

Track by calendar weeks (YYYY-WW format). Navigate forward or backward. Data organized by week.

Time Intelligence

Habits lock time slots preventing conflicts. Goals show allocated hours. Real-time clock in navigation.

Your Data Stays Yours

Local browser storage only. No servers, databases, or accounts. Work completely offline.
